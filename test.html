C. Centers

Time limit
2 seconds
Memory limit
1024 MB

Problem Statement

You are given a sequence A=(A1,A2,…,A3N) of length 3N where each of 1, 2, …, and N occurs exactly three times.

For i = 1, 2, …, N, let f(i) be the index of the middle occurrence of i in A. Sort 1, 2, …, N in ascending order of f(i).

Formally, f(i) is defined as follows.

  • Suppose that those j such that Aj = i are j = α, β, γ (α < β < γ). Then, f(i)=β.

Constraints

  • 1≤N≤105
  • 1≤Aj≤N
  • i occurs in A exactly three times, for each i = 1, 2, …, N.
  • All input values are integers.

Input

The input is given from Standard Input in the following format:

                            
N
A1 A2 … A3N

Output

Print the sequence of length N obtained by sorting 1, 2, …, N in ascending order of f(i), separated by spaces.


Examples

Sample Input 1

1
1 1 1

Sample Output 1

1

Sample Input 2

4
2 3 4 3 4 1 3 1 1 4 2 2

Sample Output 1

3 4 1 2
